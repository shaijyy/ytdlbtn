> Note: Imputnet (formerly wukko) has shut down their public API instances, so you WILL need to provide your own.  
>   
> Yet another Note: The styling for this is broken because it tries to use the now non-existing youtube styling css variables, but I don't feel like fixing it since it does not matter that much.

## Installation 
Install any userscript manager extension in your browser, such as **Tampermonkey**, **Greasemonkey** or **Violentmonkey**, then click the button below for a clean installation of the script.  
I personally use Violentmonkey, but it does not really matter.

_Support for **YouTube**, **YouTube shorts** and **YouTube music**!_

**Click** the download button to download as video (`.mp4` format), or **right click** to download as audio (`.mp3` format).

![screenshot-menu](https://raw.githubusercontent.com/shaijyy/ytdlbtn/main/images/screenshot-menu.png)

Press **P** on the keyboard to open the script's settings menu.

<b>
    <img src="https://raw.githubusercontent.com/imputnet/cobalt/main/web/static/favicon.png" alt="cobalt-icon" width=22>
    This userscript uses the <a href="https://github.com/imputnet/cobalt">Cobalt</a> API structure to download the videos, so you can hook it up to your instance and it should work just fine.  
</b>
<br>
<br>

[As stated by the Cobalt's creators](https://github.com/imputnet/cobalt/raw/main/web/static/favicon.png), this is **_NOT_** a piracy tool and cannot be used as such. It can only download free, publicly accessible content. Such content can be easily downloaded through any browser's dev tools. Pressing one button is easier, so this convenient, ad-less tool for such repeated actions was created.
